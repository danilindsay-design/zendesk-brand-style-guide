ZENDESK BRAND SYSTEM

Brand style guide

Zendesk's AI friendly digital design and development library

Overview

This document provides machine-readable access to the Greenhouse 3.0 design system for AI agents and developers. All design tokens, brand guidelines, usage rules, and component examples are embedded as structured JSON data and can be accessed programmatically.

What's included:

  • → Design Tokens: Colors, typography, spacing, breakpoints, shadows, border radius, grid system
  • → Brand Strategy: Creative platform, brand attributes, graphic language principles
  • → Usage Guidelines: Color hierarchy, button hierarchy, typography usage rules
  • → Component Examples: Production-ready code for buttons, forms, cards, and more
  • → Layout Patterns: Hero sections, feature grids, content sections with responsive examples
  • → Accessibility: WCAG 2.1 AA compliance, color contrast, keyboard navigation, ARIA labels

Use the copy buttons to extract specific sections, or access the full JSON structure at the bottom of this page.
